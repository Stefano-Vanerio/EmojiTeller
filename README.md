# EmojiTeller WebApp
Web application used to play the game "EmojTeller", based on the ChatCord App and Tweemoji Keyboard (https://repl.it/github/bradtraversy/chatcord)
## Usage
```
npm install
npm run dev

Go to localhost:3000
```
